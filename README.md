# calc
Calc is a simple C terminal dynamic expression calculator.

### Installation
```bash
git clone https://github.com/Iammethatisnotyou/calc
cd calc
```

**Build the program**
```bash
sudo make install
```

**Start the program**
```bash
calc
```

**Information**<br>
For more information check out the man page
```bash
man calc
```
